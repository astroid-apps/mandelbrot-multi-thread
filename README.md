# Mandelbrot (Multi Thread)
A touch event driven multi thread processing mandelbrot set viewer by using web workers.

移動、拡大縮小のタッチイベント（マウスイベント）に従ってWeb Workerを使用してマルチスレッドでマンデルブロ集合を描画します。
